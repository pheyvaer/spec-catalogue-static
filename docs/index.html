We use the following actions for KNoWS:

  • Contribute: KNoWS contributed to the spec in any way.
  • Create: KNoWS contributed to the spec by actively writing. This action implies the Contribute action.
  • Endorse: KNoWS didn't contribute to the spec but endorses it.
  • Lead: KNoWS supervised the creation of the spec. This action implies the Create and Contribute actions.
